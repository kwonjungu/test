# 디싹 캠프 서류 자동화

캠프명단 엑셀을 업로드하면 정해진 양식 서류를 자동 작성하는 **브라우저 단독(서버 없음)** 웹앱.
모든 처리는 브라우저 안에서만 이루어져 개인정보가 외부로 나가지 않습니다. GitHub Pages로 배포.

## 1단계 (현재): 캠프명단 → 수업신청학생등록양식 변환

캠프명단(`오전반`/`오후반` 시트)을 파싱해 익명화·형식변환 후 `수업신청학생등록양식.xlsx`로 내려받습니다.

### 변환 규칙
자세한 규칙은 [`docs/RULES.md`](docs/RULES.md) 참고.
- **학생명** 가운데 글자 `0` 익명화 (권준구→권0구)
- **연락처** 하이픈 형식 (010-8591-1183), 텍스트 서식
- **지역** 학교명 자동완성: 번들 매핑 → 원DB → NEIS API → 수동
- **학년** `초등학교 N학년` 형식 (양식 드롭다운과 동일)
- **반** 1~4 랜덤
- 드롭다운·메모·텍스트서식은 양식 템플릿을 그대로 채워 보존

### 사용법
1. `index.html`을 열거나 GitHub Pages 주소 접속
2. 캠프명단 xlsx 업로드 → **변환** → **다운로드** (오전반/오후반 파일을 각각 개별 다운로드)
3. 지역은 학교명으로 자동 입력. 미확인 학교만 "학교 검색" 또는 드롭다운으로 선택

### 지역 자동조회 (NEIS)
별도 키/설정 없이 동작합니다. NEIS 개방 API는 키 없이도 호출되고 CORS(*)도 허용되어
브라우저에서 학교명으로 전국 학교의 시·도를 바로 조회합니다.
호출량이 많아 한도에 걸릴 때만 `js/config.js`의 `NEIS_API_KEY`에 무료 키를 넣으면 됩니다(선택).

### 로컬 실행
모듈(ESM) 사용으로 `file://`이 아닌 로컬 서버가 필요합니다:
```
cd test
python -m http.server 8000
# http://localhost:8000
```

## 구조
```
index.html              UI
styles.css
js/app.js               화면/이벤트 연결
js/config.js            NEIS API 키 설정
js/convert.js           명단 파싱 · 변환 규칙 · xlsx 템플릿 채우기
js/region.js            지역(시·도) 해석기 (번들/원DB/NEIS/수동)
data/region_map.json    학교명 → 시·도 매핑 (누적)
templates/              양식 파일(xlsx) — 채우기 대상
samples/                테스트용 샘플 명단
docs/RULES.md           변환 규칙 명세
```
