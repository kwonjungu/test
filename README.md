# 디싹 캠프 서류 자동화

캠프명단 엑셀을 업로드하면 정해진 양식 서류를 자동 작성하는 **브라우저 단독(서버 없음)** 웹앱.
모든 처리는 브라우저 안에서만 이루어져 개인정보가 외부로 나가지 않습니다. GitHub Pages로 배포.

## 생성 서류

캠프명단 1개에서 클래스(오전/오후)별로 두 서류를 자동 작성합니다.

| 서류 | 형식 | 이름 | 비고 |
|------|------|------|------|
| 수업신청학생등록양식 | xlsx | **익명화**(권0구) | 드롭다운·텍스트서식 보존 |
| 수령대장 | hwpx | **실명** | 서명 대장 → 실명 필요, 차시·날짜 반영 |

### 변환 규칙
자세한 규칙은 [`docs/RULES.md`](docs/RULES.md) 참고.
- **학생명** 가운데 글자 `0` 익명화 (권준구→권0구) — 등록양식만. 수령대장은 실명.
- **연락처** `010-1234-1234` 고정 더미, 텍스트 서식
- **이메일** 임의 자동생성
- **지역** 학교명 자동완성: 번들 매핑 → 원DB → NEIS API → 수동
- **학년** `초등학교 N학년`, **반** 1~4 랜덤
- **일반학생 여부** 다문화/사회배려 학급(체크박스)에서 일반학생만 `Y`

### 사용법
1. `index.html`을 열거나 GitHub Pages 주소 접속
2. 캠프명단 xlsx 업로드 → **설정 패널에서 다문화 여부·차시·날짜 확인/수정**
3. **변환** → **등록양식(xlsx)** / **수령대장(hwpx)** 각각 개별 다운로드 (클래스별)
4. 지역은 학교명으로 자동 입력. 미확인 학교만 "학교 검색" 또는 드롭다운으로 선택
5. 원DB(모집현황)는 비PII 참고데이터(`data/program_db.json`)로 내장돼 백그라운드 자동 로드

### 지역 자동조회 (NEIS)
별도 키/설정 없이 동작합니다. NEIS 개방 API는 키 없이도 호출되고 CORS(*)도 허용되어
브라우저에서 학교명으로 전국 학교의 시·도를 바로 조회합니다.
호출량이 많아 한도에 걸릴 때만 `js/config.js`의 `NEIS_API_KEY`에 무료 키를 넣으면 됩니다(선택).

### 로컬 실행
모듈(ESM) 사용으로 `file://`이 아닌 로컬 서버가 필요합니다:
```
cd test
python -m http.server 8000
# http://localhost:8000
```

## 구조
```
index.html              UI
styles.css
js/app.js               화면/이벤트 연결 · 설정 패널
js/config.js            NEIS API 키 설정
js/convert.js           명단 파싱 · 변환 규칙 · xlsx 템플릿 채우기
js/hwpx.js              수령대장 hwpx 생성 (템플릿 셀 채우기)
js/region.js            지역(시·도) 해석기 (번들/원DB/NEIS/수동)
data/region_map.json    학교명 → 시·도 매핑 (누적)
data/program_db.json    원DB(모집현황) 비PII 참고데이터 — 백그라운드 자동 로드
templates/              양식 파일(xlsx 등록양식 · hwpx 수령대장) — 채우기 대상
samples/                테스트용 샘플 명단
docs/RULES.md           변환 규칙 명세
```

## 기술

복잡한 hwpx 처리가 필요하면 [edwardkim/rhwp](https://github.com/edwardkim/rhwp)(Rust+WASM, HWP/HWPX 읽기·쓰기) 검토.
